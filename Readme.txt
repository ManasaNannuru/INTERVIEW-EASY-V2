Steps to Run the Program

-> Go to the client folder and install node modules by using below command
            yarn install
-> Open a new window and Go to server folder and install node modules by using below command
           yarn install

Once the node modules are installed

-> go to client window and start client
           yarn start
-> go to server window and start server
          yarn start



Application is deployed on the heroku, you can access the application at the below link

https://interview-easy-v1-front-end.herokuapp.com/
